# 置信区间与裂缝蝌蚪图

## 简介

本项目包含两个 Python 函数，用于绘制置信区间图和裂缝蝌蚪图。这些函数可以帮助研究人员和数据科学家可视化模型的预测结果及其不确定性，以及地质裂缝的分布情况。
